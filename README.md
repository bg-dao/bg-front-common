# 鲁班区块链浏览器

## Build Setup

```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:11002
$ npm run dev

# build for production and launch server
$ npm run build
```

```bash
# 如果有新加的包要先
$ npm shrinkwrap --dev
# 锁定包文件
```

## linux serve command

```bash
# linux serve shell command
$ git pull
$ npm i
$ npm run build
将dist文件拷贝就行了
```
